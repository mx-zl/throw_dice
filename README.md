# throw_dice

Ich hatte in einer DAS Runde Gamemaster gemacht.
Wir hatten kein Internet und wenig Würfel also habe ich dieses Skript geschrieben um am Laptop nebenbei Würfeln zu können
Aufrufen mit:

python throw_dice.py Augenzahl Anzahl_der_Würfel
python throw_dice.py 20 3   # wirft drei 20-seitige Würfel
